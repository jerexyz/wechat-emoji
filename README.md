# wechat-emoji

[![Greenkeeper badge](https://badges.greenkeeper.io/jerexyz/wechat-emoji.svg)](https://greenkeeper.io/)
一个用以在web项目中调用微信表情的angular directive
### start use

```
git clone git@github.com:jerexyz/wechat-emoji.git
npm install
npm install -g bower
npm install -g gulp-cli
bower install
gulp serve
```
